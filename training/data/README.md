# Training Data Directory

This directory contains training datasets for the Sales Roleplay Chatbot.

## Structure:
- `conversations/` - Training conversation datasets
- `personas/` - Persona-specific training data
- `scenarios/` - Sales scenario datasets
- `feedback/` - Training feedback data

## Usage:
Training data should be placed here before running the training pipeline.

See `training/enhanced_training_coordinator.py` for training execution.
